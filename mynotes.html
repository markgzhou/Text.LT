About Text.LT

Text.LT (Light-Text) is an experiment project to implements the basic function of EverNotes. Due to the limitation on editing RTF text, we have created this application to only work on MarkDown style text.

Google Authentication is used to secure user account information as well as to protect users information from brute-force attack.

Developed by Guanlong(Mark) Zhou, and Wenqiang Bo.
